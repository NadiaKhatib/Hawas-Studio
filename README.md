# How 🙋
## Getting Started  

### Prerequisites  

###### * Clone the repo:  

SSH  🔐

```
git clone git@github.com:facg3/Hawas-Studio.git
```
https 🔐

```
git clone  https://github.com/facg3/Hawas-Studio.git
```

###### * Install the ` node.js` devDependencies:  

```  
 npm install  
```

###### * Run the Server with `Nodemon`:  

```
 npm run start:watch   
 ```

###### * Run the (module) `Tests`

```   
 npm run test  

```
